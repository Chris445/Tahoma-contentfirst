What is Tahoma?

Tahoma is a web-safe font that was developed for Microsoft by Matthew Carter and was released with Windows 95 on August 24, 1995.

It's simplistic yet bold lettering sets a solid visual aesthetic that is easy to read and grabs the viewers attention.

Tahoma was named after the Native American name for the stratovolcano Mount Rainier (Mount Tahoma), which is a prominent feature of the southern landscape around the Seattle metropolitan area.
